安装Hexo

在terminal中执行以下命令

$ sudo npm install -g hexo

初始化自己的博客项目

创建你用来保存博客的文件夹，比如：Blog 创建文件夹的命令如下

$ mkdir Blog

cd到刚创建的文件夹中

$ cd Blog

初始化hexo

$ hexo init

安装hexo工程所需包

$ npm install

启动hexo服务

$ hexo s
